# DataAnalytics_python_projects
# File: Cross selling strategy.ipynb

Objective:  
To analyze customer purchasing behavior and identify cross-selling opportunities that can help increase revenue.

Key Tasks Performed:
- Data cleaning and preprocessing
- Customer segmentation
- Product relationship analysis
- Identifying frequently bought product combinations
- Business insights and recommendations

Tools & Libraries Used:
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

Outcome: 
Provided actionable insights on which products can be recommended together to improve cross-selling strategy.



# File: HRDATA.ipynb

Objective:  
To analyze HR data and understand employee trends, performance, and attrition-related factors.

Key Tasks Performed:
- Handling missing values
- Exploratory Data Analysis (EDA)
- Analyzing employee demographics
- Identifying factors affecting employee attrition
- Visualization of HR metrics

Tools & Libraries Used:
- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

Outcome:
Helped identify important HR patterns that can assist management in improving employee retention and workforce planning.
